Node Hierarchy

Build this tree in the Godot Editor (Scene panel → right-click a node to add children):

LobbyUI                (Control)
  └─ CenterContainer   (CenterContainer)
      └─ VBoxContainer (VBoxContainer)
          ├─ Title          (Label)
          ├─ HSeparator     (HSeparator)
          ├─ AddressInput   (LineEdit)
          ├─ Buttons        (HBoxContainer)
          │   ├─ HostButton (Button)
          │   └─ JoinButton (Button)
          ├─ PlayerList     (ItemList)
          ├─ StatusLabel    (Label)
          └─ StartButton    (Button)

Step-by-Step

1 Root Node — LobbyUI
  1. Add a Control node, rename it LobbyUI.
  2. In the Inspector, set Layout > Anchors Preset to Full Rect.
  3. Attach src/ui/lobby/lobby_ui.gd as its script (Inspector → Script → drag or browse).
2 CenterContainer
  1. Add a CenterContainer child of LobbyUI, rename it CenterContainer.
  2. Set Layout > Anchors Preset to Full Rect.
3 VBoxContainer
  1. Add a VBoxContainer child of CenterContainer, rename it VBoxContainer.
  2. Inspector → Theme Overrides > Constants > Separation: 12.
4 Title Label
  1. Add a Label child of VBoxContainer, rename it Title.
  2. Inspector → Text: Project N1B.
  3. Inspector → Horizontal Alignment: Center.
5 HSeparator
  1. Add an HSeparator child of VBoxContainer. No rename needed.
6 AddressInput
  1. Add a LineEdit child of VBoxContainer, rename it AddressInput.
  2. Inspector → Placeholder Text: Host IP (leave blank for localhost).
  3. Inspector → Custom Minimum Size: x = 300.
7 Buttons HBoxContainer
  1. Add an HBoxContainer child of VBoxContainer, rename it Buttons.
  2. Inspector → Theme Overrides > Constants > Separation: 8.
  3. Add a Button child, rename it HostButton, Text: Host Game.
  4. Add a Button child, rename it JoinButton, Text: Join Game.
  5. On both buttons → Inspector → Size Flags > Horizontal: enable Expand.
8 PlayerList
  1. Add an ItemList child of VBoxContainer, rename it PlayerList.
  2. Inspector → Custom Minimum Size: x = 300, y = 120.
9 StatusLabel
  1. Add a Label child of VBoxContainer, rename it StatusLabel.
  2. Inspector → Text: leave empty.
  3. Inspector → Horizontal Alignment: Center.
10 StartButton
  1. Add a Button child of VBoxContainer, rename it StartButton.
  2. Inspector → Text: Start Game.
  3. Inspector → Disabled: ✓ checked — script enables it when lobby is full.

Signal Connections

All signal connections are made in code (lobby_ui.gd). You do not need to wire anything in the Node panel.
