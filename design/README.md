# Blue Theta Seven (Working Title)
**Genre:** Turn-Based Deckbuilder / Tabletop Skirmish Hybrid
**Status:** Pre-Prototype / System Brainstorm

## Project Overview
Blue Theta Seven is a system-first game where players build the battlefield as they play. It combines the strategic preparation of a TCG with the tactical movement of a wargame.
