# Terminology Index

This document serves as the "source of truth" for definitions used throughout Blue Theta Seven. To keep the design flexible, numerical values are noted as variables in brackets (e.g., `[Value: 10]`).

## Core Mechanics

- **Action Points (AP):** The primary resource used for almost all game interactions (Playing cards, movement, combat).
- **The Marker:** A shared track that determines whose turn it is and the current pool of AP available.
- **Income (Currency):** A secondary resource gained at the start of each turn. Primarily used for the Merchant.
- **Merchant:** A shared "market row" of items or upgrades available for purchase using Currency.

## Entities & Cards

- **Items:** Generic upgrades (weapons, armor, consumables) purchased from the Merchant and equipped to entities on the board.
- **Army Box:** A player's reserve of character entities not yet deployed.
- **Spawn Points:** Specific hexagonal tiles where characters can be deployed.
